# TxtPocket - Setup Instructions

## Prerequisites

- Flutter SDK installed (with Windows support)
- Windows 10/11 with Developer Mode enabled

## Enabling Developer Mode on Windows

This is **required** for Flutter to build Windows apps with plugins:

1. Press `Windows + I` to open Settings
2. Go to **Settings > Privacy & Security > For developers**
3. Toggle **Developer Mode** to ON
4. Confirm any prompts
5. Restart your terminal/IDE

Alternatively, run this command to open the settings directly:
```bash
start ms-settings:developers
```

## Setup Steps

1. **Install Dependencies**
   ```bash
   flutter pub get
   ```

2. **Generate Isar Files**
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```

3. **Run the Application**
   ```bash
   flutter run -d windows
   ```

## Troubleshooting

### Build Runner Issues
If you encounter issues with build_runner, try:
```bash
flutter clean
flutter pub get
flutter pub run build_runner build --delete-conflicting-outputs
```

### Windows Build Issues
- Make sure Developer Mode is enabled
- Ensure Visual Studio Build Tools are installed
- Try running `flutter doctor` to check for issues

### Database Issues
The Isar database is stored in your Documents folder. If you need to reset:
1. Close the app
2. Delete the Isar database files from `Documents\`
3. Restart the app (sample data will be reloaded)
